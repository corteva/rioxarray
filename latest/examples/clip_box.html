Clip using a bounding box

See docs for rio.clip_box:

  • DataArray.clip_box

  • Dataset.clip_box

[5]:

xdsc = xds.rio.clip_box(
    minx=-7272967.1958741,
    miny=5048602.84382404,
    maxx=-7272503.88315758,
    maxy=5049066.15654056,
)

[6]:

xdsc.plot()

<matplotlib.collections.QuadMesh at 0x7f6a9e1c64a0>

You can also clip using bounds in a CRS different from the dataset if you pass in the crs kwarg (requires rioxarray 0.12+):

[7]:

xdscn = xds.rio.clip_box(
    minx=-93.1558,
    miny=45.403,
    maxx=-93.1557,
    maxy=45.4065,
    crs="EPSG:4326",
)
