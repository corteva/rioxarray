Example - Convert dataset to raster (GeoTiff)Â¶

Often, it is desirable to take a variable (band) out of your dataset and export it to a raster. This is possible with the rio.to_raster() method. It does most of the work for you so you donâ€™t have to.

Note: The rio.to_raster() method only works on a 2-dimensional or 3-dimensional xarray.DataArray

import rioxarray
import xarray
import re
