rioxarray `rio` accessors
--------------------------

rioxarray `extends xarray <http://xarray.pydata.org/en/stable/internals.html#extending-xarray>`__
with the `rio` accessor. The `rio` accessor is activated by importing rioxarray like so:

.. code-block:: python

    import rioxarray
